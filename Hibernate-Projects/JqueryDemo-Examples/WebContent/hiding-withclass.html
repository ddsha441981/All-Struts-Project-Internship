We can map collection elements of Persistent class in Hibernate. You need to declare the type of collection in Persistent class from one of the following types: java.util.List java.util.Set java.util.SortedSet java.util.Map java.util.SortedMap java.util.Collection or write the implementation of org.hibernate.usertype.UserCollectionType The persistent class should be defined like this for collection element.
